# 🛒 Amazon Clone (CartX)

A simple **Amazon-inspired shopping website UI** built using HTML, CSS, and JavaScript. This project replicates the basic layout and user interface of an e-commerce platform, focusing on frontend design and interactivity.

---

## 🚀 Features

* 🧭 Responsive navigation bar with logo, search bar, and cart
* 🔍 Search functionality with input validation
* 🎯 Hover effects for better user experience
* 🏬 Product category sections (Fashion, Electronics, Toys, etc.)
* 🖼️ Hero section with promotional message
* ⬆️ "Back to Top" smooth scrolling feature
* 📦 Footer with multiple informational sections

---

## 🛠️ Technologies Used

* **Frontend:**

  * HTML5
  * CSS3
  * JavaScript (Vanilla JS)

* **Icons:**

  * Font Awesome CDN

---

## 💡 Functionality Details

### 🔍 Search Bar

* Displays alert if input is empty
* Shows entered search query

### 🧭 Navbar Interaction

* Dynamic border highlight on hover using JavaScript

### ⬆️ Back to Top

* Smooth scrolling to the top when clicked
